O HTML consiste em tres partes a tag "html", "head" ou cabeca e o "body" ou corpo , lembrando as tags citadas DEVEM ser passadas dentro dos sinais < > , tudo que estiver entre a tag "body" sera oque aparecera para o usuario e entre a tag "head" seram as coisas que carregaram antes de exibir pro usuario.

Atributos

(VOLTAR)

"resumidamente atributos sao propriedades que uma tag pode ter"

Exitem atributos globais e especificos, por exemplo o atributo "id" e utilizado como identificador para o elemento, o "style" que e uma tag predominantemente utilizada no css que faz com que o navegador olhe para o elemento que contem o atributo e o deixe mais "bonitinho" o atributo tambem e utilizado no HTML diretamente,tambem focado no css existe o "Class" e utilizado para englobar um grupo em uma caracteristica que voce deseja.

Falando um pouco sobre atributos que sao esoecificos de cada tag temos o "Type" por exemplo que e utilizado junto a tag input para mudar a tipagem do que sera inserido no campo por exmplo : numeros, cores ou textos, temos tambem o atributo "SRC" que utilizado junto a tag "img" o atributo e utilizado para referenciar o caminho da imgem , outros atributos utilizado com o img sao o "widht" e o "height"
